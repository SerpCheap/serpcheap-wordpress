Track where your posts, pages, products, categories and home page rank on Google — right from the WordPress admin. Powered by the serp.cheap Google Search API.

== Description ==

Add keywords to any post, page, WooCommerce product, category, or your home page and see where they rank on Google over time — with inline rank, trend sparklines, a central management dashboard, list-table columns and a dashboard widget.

* Per-post / per-product / per-category keyword tracking from the edit screen.
* Central "Rank Tracking" admin page with all keywords, current rank, 7-day change and trend.
* "Rank" column on your posts/pages/products lists.
* Dashboard widget summary (average rank, top movers, not-found count).
* Daily / weekly / manual refresh schedules + refresh-now.
* WooCommerce products & product categories supported out of the box.

= Demo build =

This build ships in **demo mode**: rankings are generated by a local mock so you can install and test the entire experience without a serp.cheap account or API key. The real connection (a secure OAuth "Connect to serp.cheap" flow that issues a per-site key) lands in the next release.

== External service ==

This plugin connects to the paid serp.cheap Google Search API (https://serp.cheap) to fetch rankings. When you track a keyword, the target URL, keyword and country are sent to https://api.serp.cheap. See the Terms (https://serp.cheap/terms) and Privacy Policy. In demo mode no data leaves your site.

== Installation ==

1. Upload the `serpcheap-rank-tracking` folder to `/wp-content/plugins/`, or install the zip via Plugins → Add New → Upload.
2. Activate the plugin.
3. Open **serp.cheap → Settings** and click **Connect (Demo)**.
4. Edit any post/page/product, add a keyword in the "serp.cheap — Rank Tracking" box, and watch the rank appear.

== Frequently Asked Questions ==

= Does this cost money? =
The plugin is free. In a live (non-demo) build each rank check consumes serp.cheap credits — pay only for what you use, no monthly minimum.

= What can I track? =
Posts, pages, WooCommerce products, categories (incl. product categories), your home page, and any custom URL.
